# Matplotlib Basics: Python Plotting Reference

A hands-on reference & cheat sheet for frequently used [Matplotlib](https://matplotlib.org/) techniques.  
Designed for practical use and step-by-step preview via Jupyter notebook.  
Includes all the essentials: lines, scatter, bars, histograms, subplots, styles, saving, annotations, and more.

## Quick Reference

- **Add grid:** `plt.grid(True)`
- **Change color / marker / line:**  
  `plt.plot(x, y, color='green', marker='o', linestyle='--')`
- **Colormaps:** Use with scatter, hist, imshow: `'viridis', 'plasma', 'coolwarm', ...'`
- **Adjust layout:** `plt.tight_layout()` avoids label overlap
