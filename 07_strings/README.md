# Strings in Python

Reference guide for string operations in Python.  
Covers core concepts: creation, concatenation, formatting, indexing, slicing, iteration, common string methods, and regular expressions. 

## Practical Patterns

- **String concatenation and formatting:** Use `+` for joining, or f-strings for readable output.
- **Splitting and cleaning:** Use `.split()` and `.strip()` for data cleansing tasks.
- **Pattern matching:** Use `re.findall()` for extracting information or searching in text.
