# Control Structures: if, while, and for Statements

A beginner-friendly reference on controlling the flow of your Python code using conditional statements and loops.  
These are foundational tools that make your code flexible, dynamic, and powerful.

## Common Pitfalls & Tips

- Indentation is critical! Indent blocks after `if`, `elif`, `else`, `while`, and `for`.
- `break` exits a loop; `continue` skips to the next iteration.
- Use `input()` for user input; always convert types as needed (e.g., `int(input(...))`).
